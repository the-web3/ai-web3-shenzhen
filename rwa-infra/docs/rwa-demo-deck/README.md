# RWA Infra — HTML Deck（离线左右翻页）

这是一份 **完全离线** 的 HTML 版“PPT Deck”，用于 Demo Day 演示：**左右翻页、目录概览、进度条、可打印导出 PDF**。

## 如何打开
- 直接用浏览器打开：`docs/rwa-demo-deck/index.html`
  - macOS：Finder 双击即可
  - 或把文件拖进 Chrome / Edge / Safari

> 建议用 Chrome/Edge 演示（兼容性最稳）。

## 操作说明
- **翻页**：
  - `←/→`：上一页/下一页
  - `PageUp/PageDown`：上一页/下一页
  - `Space`：下一页
  - `Home/End`：第一/最后一页
- **目录概览**：
  - `Esc`：打开/关闭目录概览（点选跳转）
- **触控**：
  - 手机/触控板：左右滑动翻页

## 全屏演示（推荐）
- 浏览器全屏快捷键：
  - Chrome/Edge：`⌃⌘F`（macOS）或 `F11`（Windows）

## 导出 PDF（兜底提交/备份）
1. 浏览器打开 `index.html`
2. 打印：`⌘P`（macOS）/ `Ctrl+P`（Windows）
3. 选择 “保存为 PDF”
4. 建议：布局选择 **横向（landscape）**

## 设计目标
- **评委友好**：每页最多 “1 句结论 + 3 个要点”，证据在现场直接 Demo
- **不依赖网络**：避免现场 Wi‑Fi / CDN 波动翻车
- **可审计叙事**：时间线与 hash 存证是核心说服力
