# 🌍 Ceres Protocol - AI Web3 预测市场项目提交

## 项目概述

**Ceres Protocol** 是一个基于区块链的去中心化预测市场平台，专注于气候风险和环境可持续性预测。该项目结合了人工智能和Web3技术，为用户提供了一个参与环境相关事件预测的平台。

## 🚀 演示地址

**在线演示**: https://ceres-protocol-frontend-six.vercel.app

## 🏆 项目亮点

### ✨ 核心功能

- 🌡️ **气候风险预测市场**: 7个真实的环境相关预测事件
- 🤖 **AI智能代理**: Python实现的趋势分析和预测算法
- 💰 **完整代币经济**: HSK代币 + 绿色积分奖励系统
- 🔗 **Web3集成**: 支持MetaMask钱包连接和区块链交易

### 🛠️ 技术实现

- **前端**: React + TypeScript + Tailwind CSS
- **智能合约**: Solidity + Foundry + 完整测试覆盖
- **AI服务**: Python + FastAPI + 数据分析算法
- **部署**: Vercel + HashKey Chain Testnet

### 🎨 用户体验

- 📱 响应式设计，支持桌面和移动端
- 🎨 自定义"Spring Garden"设计系统
- ⚡ 流畅的交互动画和实时数据更新
- 🔒 安全的Web3钱包集成

## 📊 项目数据

- **代码行数**: 8000+ 行
- **智能合约**: 4个核心合约
- **测试覆盖**: 单元测试 + 集成测试 + 属性测试
