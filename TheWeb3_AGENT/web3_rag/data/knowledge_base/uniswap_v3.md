# Uniswap V3 技术详解

## 什么是 Uniswap?

Uniswap 是以太坊上最大的去中心化交易所 (DEX)，使用自动做市商 (AMM) 模型。V3 版本于 2021 年 5 月发布。

## 集中流动性 (Concentrated Liquidity)

Uniswap V3 最重要的创新是集中流动性机制:

### 工作原理
- 流动性提供者可以选择自定义的价格区间
- 资金只在选定的价格范围内提供流动性
- 资本效率比 V2 提高最多 4000 倍

### 价格区间示例
假设 ETH/USDC 交易对当前价格为 2000 USDC:
- LP 可以选择只在 1800-2200 区间提供流动性
- 如果价格超出范围，流动性变为单一资产

## 手续费等级

V3 引入了多层手续费:
- 0.05%: 稳定币对
- 0.30%: 标准交易对
- 1.00%: 高波动性资产

## 预言机改进

V3 的 TWAP (时间加权平均价格) 预言机:
- 支持任意时间窗口查询
- Gas 效率提高
- 更难被操纵
