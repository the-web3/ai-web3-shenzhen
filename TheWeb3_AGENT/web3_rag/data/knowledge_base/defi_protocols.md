# DeFi 协议概览

## 什么是 DeFi?

去中心化金融 (DeFi) 是建立在区块链上的金融服务生态系统，无需传统中介机构。

## 主要 DeFi 类型

### 1. 借贷协议
- **Aave**: 支持超过 30 种资产的借贷，首创闪电贷
- **Compound**: 基于算法利率的借贷市场
- **MakerDAO**: DAI 稳定币铸造和借贷

### 2. 去中心化交易所 (DEX)
- **Uniswap**: 最大的 AMM 交易所
- **Curve**: 专注于稳定币交易，低滑点
- **SushiSwap**: 社区驱动的 DEX

### 3. 收益聚合器
- **Yearn Finance**: 自动化收益优化策略
- **Convex Finance**: Curve 生态收益增强

### 4. 衍生品协议
- **dYdX**: 永续合约交易
- **GMX**: 去中心化永续和现货交易

## 风险提示

DeFi 协议面临的主要风险:
1. 智能合约漏洞
2. 预言机操纵
3. 流动性风险
4. 监管不确定性
