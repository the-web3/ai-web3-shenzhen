# 🌱 Ceres Protocol - 黑客松演示材料

## 🎯 项目概述

**Ceres Protocol** 是一个开创性的气候风险智慧网络，通过区块链技术将专家的主观判断转化为可交易资产，并通过AI竞争机制持续提升预测质量。

### 核心创新

- 🧠 **判断资产化**: 将专家判断直接转化为链上可交易资产
- 🤖 **AI智能竞争**: AI代理自动生成竞争性预测，提升整体质量
- 💰 **激励驱动**: 绿色积分和创建者分润激励高质量预测
- 🗳️ **治理参与**: 积分持有者参与平台治理决策

## 🏗️ 技术架构

### 四合约模块化设计

```
CeresRegistry (核心注册)
├── 判断事件注册
├── 质押管理
└── 初始流动性分配

CeresMarketFactory (市场工厂)
├── 自动市场部署
├── 事件监听
└── 地址映射管理

CeresPredictionMarket (预测市场)
├── 交易执行
├── 手续费收集
└── 奖励分发

CeresGreenPoints (激励代币)
├── 积分铸造
├── 治理权重
└── 奖励管理
```

### 双市场模式

- **AMM模式**: 适合人工判断，快速启动
- **订单簿模式**: 适合AI预测，精确定价

## 🤖 AI智能代理

### 三种工作模式

1. **竞争性判断** (AMM): 监听人工事件，生成竞争预测
2. **站内趋势分析** (订单簿): 基于热度创建衍生市场
3. **站外热点捕捉** (订单簿): 基于外部数据生成预测

### 数据源集成

- 🛰️ 卫星数据 (satellite_data)
- 📰 新闻feeds (news_feeds)
- 🌐 社交媒体 (social_media)
- 🌡️ 气象API (weather_api)

## 💰 经济模型

### 核心参数

```solidity
MIN_STAKE = 0.1 HKTC          // 最低质押金额
TRADING_FEE_BPS = 200         // 交易手续费 2%
CREATOR_REWARD_BPS = 2000     // 创建者分润 20%
GREEN_POINTS_REWARD = 100     // 绿色积分奖励
```

### 激励机制

- 🎯 **正确预测奖励**: 获胜份额分配
- 💎 **绿色积分**: 每创建事件获得100积分
- 💰 **创建者分润**: 交易费的20%分润
- 🏆 **AI竞争奖励**: 额外积分奖励
