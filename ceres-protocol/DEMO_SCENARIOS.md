# Ceres Protocol 演示场景

## 🎯 演示概述

Ceres Protocol是一个开创性的气候风险智慧网络，通过区块链技术将主观判断转化为可交易资产，实现AI与人工的预测竞争。

## 🌟 核心价值主张

- **信息资产化**：将气候专家的主观判断直接转化为链上可交易资产
- **AI智能竞争**：AI代理自动生成竞争性预测，提升整体预测质量
- **激励驱动**：通过绿色积分和创建者分润激励高质量预测
- **治理参与**：积分持有者参与平台治理，推动可持续发展

## 🎬 演示场景设计

### 场景1：专家创建气候判断事件

**角色**：气候专家 Dr. Chen
**场景**：创建关于全球温度上升的专业判断

**演示步骤**：

1. **连接钱包**：展示HashKey Chain测试网连接
2. **创建判断**：
   - 事件描述："全球平均温度是否会在2030年前超过工业化前水平1.5°C？"
   - 设置价格分布：YES 65%, NO 35%（体现专家信心）
   - 质押金额：1.0 HKTC
3. **预览确认**：展示预期份额分配和奖励机制
4. **提交成功**：获得100绿色积分奖励

**展示亮点**：

- 专业判断的资产化过程
- 清晰的经济激励机制
- 用户友好的操作界面

### 场景2：AI代理自动竞争

**角色**：AI智能代理
**场景**：检测到人工判断后自动生成竞争性预测

**演示步骤**：

1. **事件检测**：AI代理监听到新的判断事件
2. **数据分析**：基于卫星数据和气候模型分析
3. **生成竞争判断**：
   - 描述："AI预测：北极海冰覆盖面积将在2026年9月达到历史最低点"
   - AI信心度：78%
   - 数据源：satellite_data
4. **订单簿创建**：自动生成5层智能订单提供流动性

**展示亮点**：

- AI与人工的智能竞争
- 多数据源融合分析
- 自动化流动性提供

### 场景3：用户参与交易

**角色**：普通投资者 Alice
**场景**：在预测市场中进行交易

**演示步骤**：

1. **浏览市场**：查看活跃的预测市场列表
2. **分析信息**：

   - 查看当前价格和交易量

3. **执行交易**：

4. **查看持仓**：在个人统计页面查看新增持仓

**展示亮点**：

- 丰富的市场信息展示
- 直观的交易体验
- 实时的持仓管理
