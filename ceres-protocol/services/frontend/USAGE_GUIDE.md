# 🌟 Ceres Protocol AI Agent Demo - 使用指南

## 📋 概述

这是一个现代化的Web前端界面，用于展示Ceres Protocol AI Agent的强大功能。支持三种AI模式的可视化演示，包括实时图表、动画效果和交互式界面。

## 🚀 快速启动

### 最简单的方式

```bash
# 进入前端目录
cd ceres-protocol/services/frontend

# Linux/macOS
./start.sh

# Windows
start.bat

# 或使用Python启动器
python3 start_demo.py
```

### 访问地址

- **主演示页面**: http://localhost:3000
- **API测试页面**: http://localhost:3000/test.html
- **API服务器**: http://localhost:8000/api/

## 🎮 演示功能

### 1. 🎯 竞争性判断模式 (AMM)

**功能**: 展示AI如何分析人类创建的预测事件并生成竞争性判断

**可视化元素**:

1. 显示人类预测事件
2. AI分析动画 (3秒)
3. 展示AI的竞争性判断
4. 价格对比图表
5. 分歧程度分析

**演示流程**:

**可视化元素**:

**演示流程**:

1. 数据源扫描动画
2. 热点事件检测

4. 市场创建条件判断

### 4. 📊 统计摘要

**功能**: 展示演示结果的统计摘要

**可视化元素**:

## 🔍 故障排除

### 常见问题

#### 1. API连接失败

**症状**: 页面显示"模拟模式"
**解决方案**:

```bash
# 检查API服务器是否运行
curl http://localhost:8000/api/status

# 重启API服务器
python3 api.py --port 8000
```

#### 2. 图表不显示

**症状**: 图表区域空白
**解决方案**:

- 检查Chart.js是否正确加载
- 清除浏览器缓存
- 检查控制台错误信息

#### 3. 动画卡顿

**症状**: 动画不流畅
**解决方案**:

- 关闭浏览器其他标签页
- 使用Chrome或Firefox最新版本
- 检查系统资源使用情况

#### 4. 移动端显示问题

**症状**: 移动设备上布局错乱
**解决方案**:

- 确保使用现代浏览器
- 检查viewport设置
- 测试不同屏幕尺寸
